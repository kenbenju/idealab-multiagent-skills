## 🛠️ Installation & Environment

### For Claude Code
1. Clone this repo: `git clone https://github.com/kenbenju/idealab-multiagent-skills.git`
2. Add to your project: `mkdir -p .claude/skills && cp -r idealab-multiagent-skills/skills/* .claude/skills/`
3. Claude will automatically detect the skills and apply the `CLAUDE.md` rules.
