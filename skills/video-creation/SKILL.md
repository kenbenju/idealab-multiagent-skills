# 🎬 Video Creation & Motion Architecture (10,000% Logic)

## 🗺️ Ontological Map (Modular Architecture)
**Hook** → **Story/Problem** → **Vector/Visual Specs** → **Motion Path** → **CTA**

---

## 📥 Inputs & 📤 Outputs

### Inputs
- **Objective:** What is the video's goal? (Conversion, Awareness, Education).
- **Core Message:** The one thing the viewer must remember.
- **Brand Assets:** Logo, colors, voice, and current DNA.

### Outputs
- **Script (Split-Column):** Audio (Voiceover) vs Visual (What we see).
- **Vector Spec List:** Description of SVGs/Vectors for the animator.
- **Motion Brief:** "Vector A moves from X to Y with easing Z."
- **Storyboard:** Sequential breakdown of frames.

---

## Instructions (Modular Logic)

### Module 1 — The Hook (0:00 - 0:03)
Specify the high-retention visual. "Vector: Central character, shock expression. Background: Neon pulse."

### Module 2 — The Technical Layer (Vectors)
Claude defines the **vectors** as functional objects:
- `v_hero`: Main brand character.
- `v_ui_overlay`: Dashboard simulation.
- `v_accent_01`: Dynamic line art for transitions.

### Module 3 — Storyboarding
Map out the **Timeline** (seconds), **Shot Type** (CU, WS, MS), and **Animation Instruction**.

```markdown
Frame 01 [0-2s]: CU of [v_hero]. Movement: Slight zoom. VO: "Tired of slow agents?"
Frame 02 [2-5s]: Split screen [v_ui_overlay] + [v_accent_01]. VO: "Meet IDEALAB..."
```
