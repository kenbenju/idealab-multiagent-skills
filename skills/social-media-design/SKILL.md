## 📜 Attention Standards (10,000% Logic)

### 1. Algorithm-Sensitive Hooking
Hooks must be tailored to how the platform detects "Retention":
- **LinkedIn:** Use a contrarian statement in the first 2 lines.
- **Instagram:** Use a "Problem/Solution" visual text overlay.
- **TikTok:** Use a "You won't believe..." or a physical pattern interrupt.

### 2. The Repurposing Matrix
Never create content for one platform only. 
- **Rule:** 1 YouTube Video -> 5 TikToks -> 3 LinkedIn Posts -> 1 Newsletter.
- **Process:** Extract the `Linguistic Anchor` and re-wrap it in the `Platform Context`.

### 3. Engagement Loops (The 'Vibe' Check)
Engagement happens in the comments. 
- **Protocol:** Generate 3 "Conversation Starters" that the user can paste as the first comment to trigger the algorithm.

### 4. Integration with Digital Twin
Test hashtags and hooks on the `digital-twin`. 
- *Logic:* If the Twin reports `Cringe Score > 10%`, the hook is "too try-hard" and must be neutralized via the `anti-bias` and `copywriting` skills.
