# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-03-25

### Added
- **13 New Skills:** (multi-agent, anti-bias, memory, token-optimization, n8n-workflows, contracts, proposals, client-onboarding, video-creation, lead-generation, email-marketing, analytics-reporting, seo-content).
- **Architecture Master:** Added `CLAUDE.md` for project-level orchestration.
- **Claude Code Support:** Added `.claude/settings.json` for native CLI integration.
- **Ontological Mapping:** Every skill now includes a logical visual map of its internal reasoning.
- **Input/Output Schemas:** Standardized data ingestion and output for all 22 skills.

### Changed
- **Format Overhaul:** Upgraded all 9 original skills to the native Claude `SKILL.md` format with YAML frontmatter.
- **README Rewrite:** Total documentation overhaul with full business cycle architecture (Mermaid).
- **Contribution Policy:** Formalized mandatory approval process for all repository changes.
- **License Update:** Upgraded to IDEALAB Partners License v2.0.
