## [1.0.0] — 2026-03-25

### Added
- 🧠 **Orchestrator Skill** — Master multi-agent router and state manager
- 📱 **Social Media Design Skill** — Content calendar, captions, visual briefs
- 🔍 **Market Research Skill** — Competitor analysis, trend mapping, ICP & SWOT
- 📋 **CRM Skill** — Contact ingestion, interaction log, pipeline & follow-up
- 📄 **Document Design Skill** — PDF reports, Excel dashboards, PowerPoint decks
- ✍️ **Copywriting Skill** — Human-toned sales pages, emails, and ads
- 🚀 **Digital Product Skill** — eBook, course, SaaS concept definition
- 🧬 **Brand DNA Skill** — Mission, vision, voice, archetype, visual system
- 🤖 **Digital Twin Skill** — Persona simulation and behavioral modeling
- Repository foundation: LICENSE, README, CONTRIBUTING, CODE_OF_CONDUCT, SECURITY
- GitHub templates: issue reports, feature requests, pull request template
